JavaScript
_____________________________________________________________________________

Функция передачи параметров будет находиться в родительском файле.
Файл JS - теста будет загружаться в iFrame файла, реализующего скрипты системы JS-трекинга.

Для передачи данных вам нужно будет вызвать родительскую JS-функцию

<button id="btnSendResult" onclick="parent.SendResult('70');  return false;">Готово</button>

Если вы хотите сигнализировать о успехе отсылки результата в систему, то
разместите поле <span id="resultElms"></span>
